# School_managment_system_coursework
These are my collage programming courseworks, one for second semester and one for third.

## Description
Simple Python programs that act as a electronic school managment system, where you can as a teacher or other staff memeber 
See, add or delete a student's data. These were created as final courseworks for second and third semester, third's one being an upgraded version of the second's, where we focused more on object oriented programming. I am open for any comments and tips. 

## Features
- Input handling
- Data processing
- Output results
- Creating/Modifying .txt files

## How to run
python S2/S3_Coursework.py
